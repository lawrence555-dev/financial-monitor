# 金控精英 (FHC-Elite) 數據指標說明文件

本文件旨在說明系統中各項數據的來源、計算邏輯以及驗證方式，以確保投資決策的透明度與準確性。

## 1. 市場報價數據
- **數據來源**：台灣證券交易所 (TWSE) 公開 OAPI。
- **更新頻率**：盤中約 5-10 分鐘，盤後提供最終成交價。
- **驗證指標**：
  - `price`: 成交價
  - `change`: 漲跌
  - `percentage`: 漲跌幅

## 2. 籌碼流向 (三大法人)
- **數據來源**：台灣證券交易所 (TWSE) [日委託買賣統計 (T86)](https://www.twse.com.tw/zh/page/trading/fund/T86.html)。
- **計算方法**：外資、投信、自營商（合計）之淨買賣超張數。
- **延遲說明**：每日約 15:30 - 16:30 由證交所發布當日數據，系統會自動抓取最近 15 個交易日之數據。

## 3. P/B 價值位階 (五年歷史位階)
- **計算公式**：
  $$ \text{目前的位階 (\%)} = \frac{\text{當前 P/B} - \text{五年最低 P/B}}{\text{五年最高 P/B} - \text{五年最低 P/B}} \times 100\% $$
- **數值定義**：
  - **P/B (Price-to-Book Ratio)**：股價 / 每股淨值。
  - **五年最高/最低**：採過去 60 個月（五年）的月底收盤 P/B 進行對比。
- **驗證方式**：可參考個股之財報公告淨值與歷史股價紀錄。

## 4. 官股行庫動態 (現階段說明)
- **狀態**：**模擬估計值**。
- **邏輯**：由於官方並未提供即時官股行庫買賣超 API，目前系統根據「官股通常與外資對作」的市場特性進行反向模擬（約佔外資買賣超的 15-30%）。
- **注意**：此數據僅供參考，**不應**作為唯一決策依據。未來計畫引入八大官股券商進出明細以提升準確度。

## 5. 散戶 FOMO 評分
- **邏輯**：近期預計導入的計算模型：
  - `(成交量變化率 + 股價爆跌後的反彈速度) / 市場平均波動度`
- **現狀**：MVP 階段採隨機模擬，僅用於展示 UI 佈局。

## 6. AI 情緒摘要
- **技術**：使用 Google Gemini 1.5 Pro。
- **輸入**：當日相關個股新聞標題、法說會精華。
- **輸出**：核心趨勢總結、市場情緒評分 (-100 ~ +100)。
- **驗證**：摘要內容會標註分析來源，用戶可點擊新聞連結進行原始內容對照。
