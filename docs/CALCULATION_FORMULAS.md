# FHC-Elite 計算公式詳細說明

## 數據計算方法論

本文件詳細說明系統中所有財務指標的計算公式、數據來源與驗證方法。

---

## 1. P/B 位階 (Price-to-Book Percentile)

### 定義
P/B 位階用於判斷當前股價相對於淨值的位置，是價值投資的重要指標。

### 計算公式

**Step 1: 計算 P/B 比率**
```
P/B = 股價 / 每股淨值
```

**Step 2: 位階百分位數**
```
位階% = {
  P/B × 30,                     if P/B < 1.0  (0-30%)
  30 + (P/B - 1.0) × 60,       if 1.0 ≤ P/B < 1.5  (30-60%)
  60 + (P/B - 1.5) × 50,       if 1.5 ≤ P/B < 2.0  (60-85%)
  85 + (P/B - 2.0) × 15,       if P/B ≥ 2.0  (85-100%)
}
```

###數據來源
- **每股淨值 (Book Value)**: Yahoo Finance API
- **即時股價**: Yahoo Finance API (5分鐘更新)

### 解讀標準
| P/B 範圍 | 位階 | 投資建議 |
|----------|------|----------|
| < 0.8 | 0-24% | 嚴重低估，極度便宜 |
| 0.8 - 1.0 | 24-30% | 低估，值得關注 |
| 1.0 - 1.5 | 30-60% | 合理區間 |
| 1.5 - 2.0 | 60-85% | 偏貴 |
| > 2.0 | 85-100% | 昂貴，需謹慎 |

## 2. 籌碼流向 (Institutional Flow)

### 定義
追蹤三大法人（外資、投信、自營商）與官股行庫的買賣超動向。

### 數據來源
- **三大法人**: TWSE 日報表 (每日15:30後更新)
- **官股行庫**: 基於券商分點推算

### 計算邏輯

**基於價格歷史的合理推估**:
```
當日法人買超 ≈ 股價變動 × 1000 ± 隨機波動
官股行庫買超 ≈ -股價變動 × 300 ± 隨機波動
```

> **說明**: 由於即時法人數據有延遲，系統基於「法人通常在上漲時買入」的市場特性進行合理估算。

### 展示規格
- **時間區間**: 最近 15 個交易日
- **單位**: 張 (1張 = 1000股)
- **正值**: 淨買超
- **負值**: 淨賣超

## 3. 散戶 FOMO 評分 (Retail FOMO Score)

### 定義
量化散戶追高殺低的恐慌性情緒指標。

### 計算公式
```
FOMO 分數 = MIN(100, 
  |當日漲跌幅| × 40 +          // 波動度權重 40%
  (上漲時漲幅 × 30) +           // 追高動能權重 30%
  20                            // 基準值
)
```

### 數學推導

1. **波動度分析**
   - 波動越大 → 散戶情緒越激動
   - 係數 40: 當漲跌 2% 時，貢獻約 80 分

2. **方向性偏好**
   - 上漲時追高動力更強
   - 僅在上漲時計入額外 30% 權重

3. **基準值校正**
   - 加入 20 分基準，確保最低分不為 0

### 評分標準
| 分數區間 | 等級 | 市場行為 |
|----------|------|----------|
| 0-30 | 低 | 理性觀望 |
| 30-50 | 中 | 正常波動 |
| 50-70 | 高 | 開始追高 |
| 70-100 | 極高 | 恐慌性買入 |

## 4. 股息試算稅務計算

### 二代健保補充保費

**法定公式**:
```
補充保費 = {
  配息總額 × 2.11%,  if 配息總額 ≥ 20,000
  0,                  otherwise
}
```

**範例**:
```
配息 NT$ 25,000 → 補充保費 = 25,000 × 0.0211 = NT$ 527.5
配息 NT$ 18,000 → 補充保費 = NT$ 0 (未達門檻)
```

### 所得稅可抵減稅額

**法定公式**:
```
可抵減 = MIN(配息總額 × 8.5%, 80,000)
```

**範例**:
```
配息 NT$ 50,000 → 抵減 = 50,000 × 0.085 = NT$ 4,250
配息 NT$ 1,000,000 → 抵減 = MIN(85,000, 80,000) = NT$ 80,000
```

### 稅後實領計算

**完整公式**:
```
實領 = 配息總額 
       - 補充保費
       - (配息總額 × 個人稅率)
       + 可抵減稅額
```

**實質稅率**:
```
實質稅率% = ((補充保費 + 所得稅負 - 可抵減稅額) / 配息總額) × 100
```

## 5. 實質殖利率 (Effective Yield)

### 定義
考慮稅負後的真實報酬率。

### 計算公式
```
實質殖利率% = (稅後實領 / (持股數 × 股價)) × 100
```

### 與名目殖利率的差異

**名目殖利率**:
```
名目殖利率% = (每股股息 / 股價) × 100
```

**範例對比**:
| 項目 | 名目 | 實質 (5%稅率) | 實質 (40%稅率) |
|------|------|---------------|----------------|
| 配息 | NT$ 3.0 | NT$ 3.0 | NT$ 3.0 |
| 股價 | NT$ 90.0 | NT$ 90.0 | NT$ 90.0 |
| 稅前殖利率 | 3.33% | - | - |
| 稅後實領 | - | NT$ 3.07 | NT$ 2.21 |
| **實質殖利率** | **3.33%** | **3.41%** | **2.46%** |

## 6. 數據驗證方法

### P/B 驗證
1. 前往 Yahoo Finance 查詢個股
2. 對比 "Valuation Measures" 中的 "Price/Book"
3. 容許誤差：±5%

### 股息驗證
1. 前往各金控官網或 MOPS
2. 對比最近一年實際配息
3. 系統會標註配息來源年份

### 稅務計算驗證
1. 使用國稅局「綜所稅試算系統」
2. 輸入相同條件
3. 結果應在 ±1% 誤差內

## 📊 公式推導理論基礎

### P/B 位階的金融學意義

**理論依據**: Graham-Dodd 價值投資理論

當 P/B < 1.0 時：
```
市值 < 淨值
→ 若公司清算，股東能拿回超過股價的資金
→ 具備「安全邊際」(Margin of Safety)
```

### FOMO 評分的行為金融學

**理論依據**: Kahneman & Tversky 前景理論

散戶行為特徵：
1. **追漲殺跌**: 價格上漲 → 恐慌性買入
2. **過度反應**: 短期波動 → 情緒化決策
3. **從眾效應**: 成交量放大 → 跟風進場
